# scentedcandle
<h3>一、開發動機</h3>
現代人生活壓力大，每天都在接收過多的資訊量，甚至開始對身心造成影響，像是社群恐慌症(FOMO)，為一種由患得患失所產生持續性的焦慮，得上這種症的人總會感到別人在自己不在時經歷了什麼非常有意義的事情，容易導致焦慮。若能找到適合自己的放鬆方法、暫時與世界脫離，why not?<br>
香氛，能夠舒解壓力、放鬆緊繃的神經，甚至擁有一個良好品質的睡眠。好的香氛聞起來能夠使人身心舒暢、徹底放鬆，而卸下心靈的疲憊，沉浸在自己喜歡的香氛情境裡，暫時忘記煩惱的人事物，香氛蠟燭不僅是個好的選擇，更可以提高生活品質！<br>
我們希望透過此網頁匯集高品質、多種類的蠟燭，可讓使用者省去查找資訊的麻煩，並且可以一目了然香氛蠟燭的品牌、價格，而提供此平台讓香氛蠟燭愛好者可以互相交流，幫助大家一起找到適合的香氛蠟燭，在這煩躁的世界中繼續苟延殘喘，從煩躁的高壓生活中喘口氣。

<h3>二、系統特點</h3>
本網站會提供多種品項的香氛蠟燭來讓使用者進行閱覽，並且設計友善的使用者介面，讓使用者在第一次使用就可以上手，在首頁也有顯示相關主題文章提供閱讀，幫助大家找到靈感。<br>
系統不管是否為會員都允許大家來進行點閱，但是註冊成為會員之後才會提供並享有更多的功能可以使用，例如將喜愛的產品加入清單內，而且當使用者所選的產品有誤時，都可以再進行修改，也會在清單中顯示挑選的商品供使用者查詢檢閱。本網站的特點為任何人都可以註冊加入會員，不需要繁複的程序，只要填寫相關資料即可成為會員，共同分享與交流。

<h3>三、系統功能</h3>
主要分為登入前與登入後，大部分的功能都是登入後才能使用，而若是沒有帳戶也可以透過註冊，並登入後來操作功能。<br>
<b>登入前</b>，能夠從首頁查看其他相關文章，也可以在產品的分頁看商品、留言，最後可以選擇登入，而沒有會員的也可以透過註冊以登入來使用會員擁有功能。<br>
<b>登入後</b>， 可以透過產品的分頁來選擇產品，並新增至清單內，而於清單內也可以，刪除所選的產品，並可以留言想說的話，在會員管理的方面，則可以修改會員資料，像是密碼、姓名、手機號碼、地址、信箱、性別以及生日，最後登出，系統也會關閉會員才能使用的功能。<br>
